## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- PostgreSQL
- Stripe API key

### Installation

    git clone https://github.com/aungsoe-lin/foodx-res.git
    cd foodx-res
    npm install
    cp .env.example .env  # Add your API keys
    npm run dev
